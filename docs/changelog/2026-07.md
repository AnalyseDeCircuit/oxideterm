## 2026-07-05: OxideSens tool reply guardrail cleanup (v1.6.11)

### Fixed

- Removed the over-strict OxideSens Evidence Binding reply replacement that could hide valid tool-backed answers behind a generic tool-evidence warning.
- Stopped asking models to emit `<evidence_claims>` blocks and stripped any leftover evidence blocks from visible chat replies.
- Fixed legacy Default execution profiles overriding the global AI tool-calling switch after users enabled tools in settings.

### Notes

- This is a patch release. No user data migration is required.

## 2026-07-04: Confirm dialog interaction lock fix (v1.6.10)

### Fixed

- Fixed a Tauri WebView interaction lock that could leave the app visible but unclickable after confirming destructive actions such as deleting an imported SSH connection.
- Kept the confirmation dialog mounted through its close transition so Radix can release its global pointer lock before the app refreshes connection state.

### Notes

- This is a patch release. No user data migration is required.
